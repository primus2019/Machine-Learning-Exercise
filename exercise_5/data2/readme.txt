fibroblast.csv: 3454 genes by 2119 cells
myofibroblast.csv: 3454 genes by 178 cells
All values are gene expression values measured with the SMRT-seq2 protocol. No pre-processing. 
Data are from mouse heart samples of the Tabula Muris project. 
fibroblast.csv contains the cells annotated as the cell type Fibroblast, and myofibroblast.csv contains the cells annotated as the cell type myofibroblast.
